**Overview**

This project demonstrates an end-to-end data analytics workflow, covering data ingestion, exploratory analysis, data cleaning, SQL-based analysis, and business-focused visualization. The goal is to transform raw customer shopping data into actionable insights using industry-standard tools and clear storytelling.

The project is designed to reflect real-world analytics work and is suitable for roles such as Data Analyst, Business Analyst, or BI Analyst.

**Dataset**

Type: Customer shopping / transactional data

Format: CSV

Content Includes:

Customer demographics

Purchase behavior

Product categories

Transaction details

Spending patterns

(Dataset used for educational and analytical purposes.)

**Tools & Technologies**

Python: Data loading, cleaning, and EDA

Pandas, NumPy, Matplotlib, Seaborn

SQL Server: Data storage and analytical queries

Power BI: Interactive dashboards and visualizations

Gamma: Report and presentation (PPT) creation

Jupyter Notebook: Analysis and documentation

**Project Workflow / Steps**

**1. Data Loading**
Imported dataset into Python using Pandas

Performed initial inspection and schema validation

**2. Exploratory Data Analysis (EDA)**

Analyzed distributions, trends, and outliers

Identified key metrics and behavioral patterns

Visualized spending, frequency, and category trends

**3. Data Cleaning**

Handled missing values and inconsistencies

Standardized column formats and data types

Prepared clean, analysis-ready dataset

**4. SQL Analysis (SQL Server)**

Loaded cleaned data into SQL Server

Wrote SQL queries to:

Analyze customer segments

Identify top products and categories

Track revenue and purchase trends

**5. Power BI Dashboard**
Built an interactive dashboard with:

KPIs (Revenue, Customers, Transactions)

Category-wise and customer-level insights

Filters for dynamic analysis

**6. Reporting & Presentation**

Created a structured analytical report

Designed a professional PPT using Gamma

Focused on insights, trends, and business recommendations

**Dashboard**

The Power BI dashboard provides:

High-level KPIs for quick decision-making

Customer and product-level drill-downs

Visual trends for spending and purchasing behavior

**Results & Insights**

Identified high-value customer segments

Highlighted top-performing product categories

Uncovered purchasing trends and seasonal behavior

Provided data-driven insights to support business decisions
